# 🧠 Smart AI Route Planner and Navigation Simulator

An interactive AI-powered route planner and pathfinding visualizer built using **Python**, **CustomTkinter**, and classic **Artificial Intelligence search algorithms**.

This project demonstrates how intelligent agents find optimal paths in a grid environment while providing real-time visualization of the search process.

## ✨ Features

- Interactive grid editor
- Place Start and Goal nodes
- Add and remove obstacles
- BFS (Breadth First Search)
- Animated search visualization
- Animated shortest path
- Modern CustomTkinter interface
- Expandable architecture for multiple AI algorithms

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/rohitlaxman/smart-ai-route-planner.git
```

Go into the project

```bash
cd smart-ai-route-planner
```

Create a virtual environment

```bash
python -m venv .venv
```

Activate it

Windows

```bash
.venv\Scripts\activate
```

Install dependencies

```bash
pip install customtkinter networkx matplotlib numpy pillow
```

Run

```bash
python main.py
```
